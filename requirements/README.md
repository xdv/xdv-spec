# XDV Requirements Index

This directory stores frozen requirement metadata for XDV-001..XDV-084.

## Files

- `xdv_requirement_index.csv`: machine-readable canonical requirement index with status tags and criteria summaries.
- `xdv_requirement_pass_fail.md`: human-readable pass/fail criteria matrix.

## Update Rules

- Do not modify frozen status tags or requirement semantics without freeze-version bump.
- Any published requirement change must update both CSV index and criteria matrix in the same change set.
